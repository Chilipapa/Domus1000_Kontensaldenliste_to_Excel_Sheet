Domus1000 Kontensaldenliste to Excel Sheet

Kurzbeschreibung
----------------
Skript zum Extrahieren und Verarbeiten von Kontensaldenlisten (z.B. für Domus1000) und Export/Analyse in Excel/Word-Formate.

Schnellstart (virtuelle Umgebung)
---------------------------------
Im Projekt-Root ausführen:

python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt

Ausführen
---------
Nach Aktivierung der venv:

python extract_accounts.py

Dateien
-------
- `extract_accounts.py` – Hauptskript
- `requirements.txt` – Abhängigkeiten
- `HOWTO_VENV.txt` – kurze Anleitung zur venv

Hinweis
------
Die virtuelle Umgebung `.venv/` ist in `.gitignore` und wird nicht ins Repository hochgeladen.
